# CRSkip Installation

## 1. Clone the Repository

Open a terminal and run:

```bash
git clone https://github.com/DevilsN3st/CR-skip
cd crskip
```

Replace the URL with the actual repository URL.

---

## 2. Open Chrome Extensions

Open:

```text
chrome://extensions
```

---

## 3. Enable Developer Mode

Turn on **Developer mode** using the toggle in the top-right corner.

---

## 4. Load the Extension

1. Click **Load unpacked**
2. Select the cloned project folder
3. The extension will be added to Chrome

---

## 5. Start Using CRSkip

1. Open Crunchyroll and play an episode
2. Click the CRSkip extension icon
3. Enable **Auto-Skip**
4. Select the skip types you want enabled

CRSkip will automatically click supported skip buttons during playback.

---

## Updating

To get the latest changes:

```bash
git pull origin main
```

After pulling updates:

1. Open `chrome://extensions`
2. Click **Reload** on the CRSkip extension
3. Refresh any open Crunchyroll tabs
